Projects

Fast Matrices

FastMatrices is a GPU accelerated matrix library written in C# with the help of ILGPU.
It support many different methods of operating on matrices, including single-threaded,
multi-threaded and of course GPU accelerated "operators" for maximum versatility.
Read more about it in the README.md.

Paper-Auto-Updater

Paper-Auto-Updater is a plugin for the widely popular minecraft server software
Paper. The plugin is very simple. All it does is update the server software to the newest
version when shutting down. It can also roll back or update to specific versions
with a simple command interface. It's not my best work, and some of the design
descisions are... questionable, but it works. Read more about it in the README.md.

PingMc

This is yet another minecraft related piece of software. This time, however,
it's not a plugin for a server. PingMc is a discord bot written in Go that
pings minecraft servers for information about player counts. It then updates
a channel's name based on that information. Read more about it in the README.md.

This bot is currently a little broken and therefore not active.

Contributions

ILGPU

ILGPU is a .NET library that allows programmers to write GPU accelerated code
without resorting to low-level languages like OpenCL C or CUDA C. I have
contributed a miniscule amount to this library, but I wanted to include it here as
it's a very cool project and the devs are very kind and welcoming. I encourage you
to check it out and contribute! Read more about it in the README.md.
